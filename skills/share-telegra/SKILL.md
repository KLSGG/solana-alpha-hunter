# Share Telegra Skill

## Description
This skill enables the agent to publish long text or Markdown content to Telegra.ph, creating a clean web link that supports Telegram's Instant View.

## Usage
```
share-telegra <content_or_file_path> [title]
```

## Output
Returns the URL of the published Telegra.ph page.
